
# 🚀 Margadarshak AI — AI Career Guidance Platform

Margadarshak AI is an **AI-powered career guidance platform** that helps students and professionals discover their career path, build skills, practice interviews, and track progress.

---

#  Features

##  AI Career Assessment
- Dynamic AI questions
- Career recommendation
- Skill gap analysis

##  AI Career Roadmap
- Personalized roadmap
- Learning timeline
- Project recommendations

##  Resume Analyzer
- Resume score
- ATS score
- Skill detection
- AI suggestions

## AI Interview Simulator
- Voice-based interview
- Real-time feedback
- AI mock interviews

##  English Mentor
- Live conversation AI
- Daily speaking challenge
- Voice mentor
- Progress tracking

##  Dashboard
- Career progress
- Resume score
- Interview score
- Skills tracking

##  Team Finder
- Create teams
- Join teams
- Skill-based matching

##  Community
- Posts
- Comments
- Likes
- Discussions

##  Company Tracker
- Track job applications
- Interview stages
- Offers

#  Installation

## 1. Clone Repository

```bash
git clone https://github.com/yourusername/margadarshak-ai.git
cd margadarshak-ai
2. Install Dependencies
Bash
npm install
3. Create .env File
Create .env in root folder
Environment
PORT=3000

# AI
OPENAI_API_KEY=
GROQ_API_KEY=

# Supabase
SUPABASE_URL=
SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=

# Auth
JWT_SECRET=
SESSION_SECRET=

# Voice
ELEVENLABS_API_KEY=
4. Run Project
Bash
npm run dev
